================================================================================
                          WHAT YOU NEED TO DO
================================================================================

⏭️  STEP 1: Get MongoDB Atlas Credentials
   1. Go to: https://www.mongodb.com/cloud/atlas
   2. Sign up (free tier available)
   3. Create a free M0 cluster
   4. Get your connection string: mongodb+srv://user:pass@cluster.net/db?...

⏭️  STEP 2: Update .env File
   Edit: /workspaces/neofuture_hackathon/.env
   
   Replace these lines with YOUR MongoDB credentials:
   
   MONGO_URI=mongodb+srv://YOUR_USERNAME:YOUR_PASSWORD@YOUR_CLUSTER.mongodb.net/talentiq?retryWrites=true&w=majority
   FLASK_SECRET_KEY=your_secure_key_here

⏭️  STEP 3: Start the App
   
   Option A (Recommended):
   $ ./RUN_APP.sh
   
   Option B:
   $ cd /workspaces/neofuture_hackathon
   $ python3 app.py
